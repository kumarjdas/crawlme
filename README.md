**The Ultimate Food Crawl Planner**

Crawlme helps you plan the perfect food crawl with your friends. Simply choose a starting location, a food type (e.g., "Tacos", "Burgers", "Pizza"), and how many stops you want. Crawlme will find the best-rated spots within your radius, optimize the route, and give you a shareable plan.

## Features

-   **Smart Search**: Finds top-rated places using Google Places API (New).
-   **Route Optimization**: Calculates the most efficient driving route between stops.
-   **Interactive Map**: Visualize your crawl on a map.
-   **Mobile Friendly**: Fully responsive design for planning on the go.
-   **Customizable**: Reorder stops, add specific places, and adjust search radius.
-   **Shareable**: Share your crawl configuration with a simple link.

## Technologies

-   React
-   Vite
-   Google Maps Platform (Maps JS API, Places API, Directions API)
-   Framer Motion
-   Lucide React

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Create a `.env` file with your `VITE_GOOGLE_MAPS_API_KEY`.
4.  Run the dev server: `npm run dev`
